# Welkom op mijn website

Dit is de homepage van mijn Hugo website. Hugo is een snelle en flexibele static site generator geschreven in Go.

## Waarom Hugo?

- **Snel**: Hugo bouwt je site in milliseconden
- **Flexibel**: Volledig aanpasbaar met templates
- **Modern**: Ondersteuning voor markdown, shortcodes en meer
- **Geen database nodig**: Alles is statische HTML
